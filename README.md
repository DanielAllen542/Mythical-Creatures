# Mythical Creatures

## Installation Steps

1. Clone the repo
2. cd into the directory
3. Run your first test suite ```npm test test/unicorn-test.js```
4. Make the tests pass
5. Lather, rinse, repeat through the creatures (in order), running tests and making things pass

## Creature Order

* `unicorn`
* `vampire`
* `dragon`
* `hobbit`
* `pirate`
* `wizard`
* `medusa`
* `werewolf`
* `centaur`
